It's recommended to compress  "cute.transitions.all.js" with Javascripr packer (Base62 encode) and others with Microsoft Ajax Minifire or YUI Compressor.
The "cute.transitions.all.js" contains all of the transitions, you can create custom transition list with transition gallery.

Javascript Packer : http://dean.edwards.name/packer/
YUI Compressor  : http://developer.yahoo.com/yui/compressor/  or  http://yui.2clics.net/
MS Ajax Minifire  : http://ajaxmin.codeplex.com/ or http://sundgaard.dk/javascript-minify.aspx
